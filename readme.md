# Build instructions (Windows)

Step 1: Download Love2D

Step 2: Download the ZIP file of this repo.

Step 3: Rename zip file to love file.

Step 4: Merge the love file with Love2D exe through CMD. | Optional

Step 5: Add necessary DLLs. | Optional

# Build instructions (Android)
